# 校园淘 - 实施计划（修订版）

## 项目概述
校园信息撮合平台，提供二手交易、拼团团购、校园论坛三大功能。

**技术栈**: React 19 + Tailwind CSS 4 + TypeScript + Vite SPA + React Router v7

**核心原则**: 以页面为单位打磨，完成一个再继续下一个。每个页面必须交付完整交互，不做半成品。

---

## 全局交互补全

以下交互跨页面，需要在组件层面统一处理：

| 交互 | 涉及组件 | 说明 |
|------|---------|------|
| 收藏切换 | ProductCard + SecondhandDetail | 心形填充/线框切换，本地状态 |
| 点赞切换 | PostCard + ForumDetail | 计数+1/-1，图标切换 |
| 分享功能 | 所有详情页 | 复制链接到剪贴板 + toast 提示 |
| Toast 提示 | 全局 | 需要一个简单的 Toast 组件 |

## 执行顺序

```
页面 1 首页 → 页面 2 二手列表 → 页面 3 二手详情 →
页面 4 拼团列表 → 页面 5 拼团详情 → 页面 6 论坛列表 →
页面 7 论坛详情 → 页面 8 个人中心 → 页面 9 编辑资料 →
页面 10 聊天 → 页面 11 404
```

**每完成一个页面，确认所有交互可用后再继续下一个。**
